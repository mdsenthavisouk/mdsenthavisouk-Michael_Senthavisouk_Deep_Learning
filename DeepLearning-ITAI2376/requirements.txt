    "## Part 2 \u2014 The Q-Learning Agent: What Learning Looks Like\n",
    "\n",
    "Now we train a Q-Learning agent. Unlike the random agent, this one **learns from experience**: after each action it takes, it updates its understanding of which actions are valuable in which situations.\n",
    "\n",
    "### How Q-Learning Works (Plain English)\n",
    "\n",
    "Q-Learning maintains a **Q-table** \u2014 essentially a lookup table that answers one question:\n",
    "*\"Given the current situation I'm in, how valuable is each possible action?\"*\n",
    "\n",
    "At the start, every entry in this table is 0 \u2014 the agent knows nothing. As it takes actions and observes what happens, it updates the table: if an action led to a good outcome (pole stayed up), that entry goes up. If it led to a bad outcome (pole fell), it goes down. Over many episodes, the agent builds up a reliable guide for what to do in each situation.\n",
    "\n",
    "**The exploration/exploitation tradeoff:**\n",
    "\n",
    "The agent uses a parameter called **epsilon (\u03b5)** to balance two competing needs:\n",
    "\n",
    "- **Explore:** Try random actions to discover new strategies (high epsilon = more randomness)\n",
    "- **Exploit:** Use what you've already learned (low epsilon = follow the Q-table)\n",
    "\n",
    "Early in training, epsilon is high \u2014 the agent explores a lot. Over time, epsilon decays \u2014 the agent increasingly trusts its learned Q-table. This is called **epsilon-greedy exploration**.\n",
    "\n",
    "**Why not just exploit from the start?**\n",
    "\n",
    "If the agent always did what it currently thinks is best, it would never discover that there is something even better. A salesperson who always calls the same client because that client once bought something will never discover better prospects. Exploration is necessary for learning.\n",
    "\n",

    "# Q-LEARNING AGENT \u2014 FULL IMPLEMENTATION\n",
    "# Read the comments. You don't need to modify this to complete\n",
    "# the lab \u2014 but understanding what each part does is the point.\n",
    "# \u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\u2500\n",
    "\n",
    "class QLearningAgent:\n",
    "    \"\"\"\n",
    "    A Q-Learning agent for CartPole.\n",
    "\n",
    "    CartPole's state space is continuous (cart position, velocity, etc.)\n",
    "    but Q-tables require discrete states. We solve this by 'binning' \u2014\n",
    "    dividing the continuous range of each variable into buckets.\n",
    "    Think of it like rounding a temperature to the nearest 5 degrees\n",
    "    instead of tracking every decimal.\n",
    "    \"\"\"\n",
    "\n",
    "    def __init__(self, n_bins=10, learning_rate=0.1, discount=0.99,\n",
    "                 epsilon=1.0, epsilon_decay=0.995, epsilon_min=0.01):\n",
    "        \"\"\"\n",
    "        Parameters:\n",
    "          n_bins        : how many discrete buckets to divide each continuous observation\n",
    "          learning_rate : how fast the agent updates its Q-values (too high = unstable,\n",
    "                          too low = learns slowly)\n",
    "          discount      : how much the agent values future rewards vs. immediate\n",
    "                          (0 = only care about now, 1 = equally value now and future)\n",
    "          epsilon       : starting exploration rate (1.0 = fully random at the start)\n",
    "          epsilon_decay : how quickly exploration decreases each episode\n",
    "          epsilon_min   : the lowest epsilon will ever go (always some exploration)\n",
    "        \"\"\"\n",
    "        self.n_bins = n_bins\n",
    "        self.lr = learning_rate\n",
    "        self.discount = discount\n",
    "        self.epsilon = epsilon\n",
    "        self.epsilon_decay = epsilon_decay\n",
    "        self.epsilon_min = epsilon_min\n",
    "\n",
    "        # Define the range of each state variable so we can bin it\n",
    "        # Order: [cart_pos, cart_vel, pole_angle, pole_angular_vel]\n",
    "        self.obs_low  = np.array([-2.4, -3.0, -0.25, -3.5])\n",
    "        self.obs_high = np.array([ 2.4,  3.0,  0.25,  3.5])\n",
    "\n",
    "        # The Q-table: 4 state dimensions \u00d7 n_bins each, 2 possible actions\n",
    "        # Initialized to zeros \u2014 the agent knows nothing at the start\n",
    "        self.q_table = np.zeros([n_bins] * 4 + [2])\n",
    "\n",
    "    def discretize(self, state):\n",
    "        \"\"\"Convert a continuous state vector into discrete bin indices.\"\"\"\n",
    "        bins = np.linspace(self.obs_low, self.obs_high, self.n_bins + 1)[1:-1].T\n",
    "        return tuple(np.digitize(state[i], bins[i]) for i in range(4))\n",
    "\n",
    "    def choose_action(self, state):\n",
    "        \"\"\"\n",
    "        Epsilon-greedy action selection:\n",
    "        - With probability epsilon: explore (pick a random action)\n",
    "        - With probability (1 - epsilon): exploit (pick the best known action)\n",
    "        \"\"\"\n",
    "        if np.random.random() < self.epsilon:\n",
    "            return np.random.randint(2)               # Explore: random action\n",
    "        return np.argmax(self.q_table[state])          # Exploit: best known action\n",
    "\n",
    "    def learn(self, state, action, reward, next_state, done):\n",
    "        \"\"\"\n",
    "        Q-Learning update rule (in plain English):\n",
    "\n",
    "        The Q-value for (state, action) should equal:\n",
    "          the reward we just got\n",
    "          + the discounted best future reward we expect from next_state\n",
    "\n",
    "        We move the current Q-value a small step (learning_rate) toward\n",
    "        this target. Repeat millions of times, and the Q-table converges\n",
    "        to accurate estimates of action value.\n",
    "        \"\"\"\n",
    "        current_q = self.q_table[state][action]\n",
    "\n",
    "        if done:\n",
    "            target = reward       # No future reward if the episode ended\n",
    "        else:\n",
    "            target = reward + self.discount * np.max(self.q_table[next_state])\n",
    "\n",
    "        # Update: move current Q-value toward the target\n",
    "        self.q_table[state][action] += self.lr * (target - current_q)\n",
    "\n",
    "    def decay_epsilon(self):\n",
    "        \"\"\"Reduce exploration rate after each episode (explore less as we learn).\"\"\"\n",
    "        self.epsilon = max(self.epsilon_min, self.epsilon * self.epsilon_decay)\n",
    "\n",
    "\n",
    "print(\"\u2705 QLearningAgent class defined.\")\n",
    "print(\"   Q-table shape:\", QLearningAgent().q_table.shape)\n",
    "print(\"   Total Q-values to learn:\", QLearningAgent().q_table.size)"

    "\n",
    "def train_agent(epsilon_decay, label, episodes=500):\n",
    "    env = gym.make('CartPole-v1')\n",
    "    agent = QLearningAgent(epsilon_decay=epsilon_decay)\n",
    "    scores = []\n",
    "    for _ in range(episodes):\n",
    "        state, _ = env.reset()\n",
    "        state = agent.discretize(state)\n",
    "        total, done = 0, False\n",
    "        while not done:\n",
    "            action = agent.choose_action(state)\n",
    "            ns, r, term, trunc, _ = env.step(action)\n",
    "            done = term or trunc\n",
    "            ns_disc = agent.discretize(ns)\n",
    "            agent.learn(state, action, r, ns_disc, done)\n",
    "            state, total = ns_disc, total + r\n",
    "        agent.decay_epsilon()\n",
    "        scores.append(total)\n",
    "    env.close()\n",
    "    final_avg = np.mean(scores[-50:])\n",
    "    print(f\"  {label:40s} \u2192 last-50 avg: {final_avg:.1f}\")\n",
    "    return scores\n",
    "\n",
    "print(\"Training comparison agents (this takes about 2\u20133 minutes)...\\n\")\n",

    "\n",
