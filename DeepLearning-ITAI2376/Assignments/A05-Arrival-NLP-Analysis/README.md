# A05 — Arrival NLP Analysis (Submission Artifact)

## Overview

In this assignment, I analyzed the film **Arrival** through a Natural Language Processing lens. My central focus was on how language structure influences cognition, and how that idea connects to AI communication design.

Although the filename is labeled as **A05**, the document content itself is titled **A04: Analyzing "Arrival" Through the Lens of NLP**. I kept that original context intact while documenting the work professionally.

## My Approach

I used a conceptual NLP framing rather than code-heavy implementation. I explored:

- How non-linear representations of meaning challenge traditional token-by-token processing
- Why methods like stemming, lemmatization, n-grams, and TF-IDF become difficult when language does not follow linear sequence assumptions
- How context, symbol structure, and temporality affect interpretation
- The bridge between human language constraints and AI model design

## Key Insights I Documented

- **Representation matters:** If meaning is encoded holistically (circular/non-linear), standard sequential NLP pipelines can lose context.
- **Time and language are linked:** Human communication is usually time-sequenced, but AI systems can process information without the same temporal constraints.
- **Feature extraction assumptions can break:** Common text techniques depend on ordered, segmentable language units.
- **AI communication design opportunity:** New encoding/representation strategies could unlock better reasoning in future language models.

## What I Learned

This assignment helped me think beyond implementation and focus on first principles:

- Why model assumptions should be questioned before choosing an NLP technique
- How symbolic structure shapes what a model can learn
- How interdisciplinary thinking (film + linguistics + AI) can reveal meaningful design ideas for real AI systems
