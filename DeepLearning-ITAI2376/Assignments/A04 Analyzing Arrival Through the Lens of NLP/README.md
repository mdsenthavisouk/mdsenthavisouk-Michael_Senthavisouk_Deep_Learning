# Arrival NLP Analysis

## Overview

In this assignment, I analyzed the film **Arrival** through a Natural Language Processing lens. My main focus was on how language structure influences cognition and how that connects to AI communication design.

## My Approach

I used a conceptual NLP framing and explored:

- How non-linear representations of meaning challenge token-by-token processing
- Why methods like stemming, lemmatization, n-grams, and TF-IDF struggle when language is not strictly sequential
- How context, symbol structure, and temporality affect interpretation
- How insights from human language can inform AI model design

## Key Insights I Documented

- **Representation matters:** If meaning is encoded holistically, standard sequential NLP pipelines can lose context.
- **Time and language are linked:** Human communication is often time-sequenced, while AI systems can process information with different temporal assumptions.
- **Feature extraction assumptions can break:** Common text techniques depend on ordered, segmentable language units.
- **AI communication design opportunity:** New representation strategies could support stronger reasoning in future language models.

## What I Learned

This assignment helped me focus on first principles:

- Question model assumptions before selecting an NLP technique
- Understand how symbolic structure limits or expands what a model can learn
- Use interdisciplinary thinking (film, linguistics, and AI) to develop better design ideas
